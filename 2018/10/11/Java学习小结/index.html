5. &, && 的区别 (同|, || 的区别)

  1. 普通与 & 比较所有条件, 而短路与 && 在遇到某条件为false时直接返回false
  2. 位运算只有普通与 & 和普通或 | 才可以参与

五、 方法的定义与使用

1.方法的重写与重载

  • 重写(Override)：又称“覆盖”。子类继承了父类的方法并重新进行实现，返回值，函数名与形参都不变且访问权限不能比父类中被重写的方法的访问权限更低

  • 重载(Overload)：同一个类中，方法名相同，参数列表不同，返回值类型可相同可不同

六、 面向对象

1. 面向对象三大特点：

  • 封装性： 保护内部结构的安全性
  • 继承性： 在原有代码上扩充新的功能
  • 多态性： 同一个行为具有多个不同表现形式

2. 堆内存与栈内存

  • 堆内存： 保存对象的属性内容，需要用关键字new来开辟
  • 栈内存： 保存的是一块堆内存的地址

七、 类与对象

1. 构造方法

  • 开发建议：类之中可以有多个构造方法，但是必须保留一个无参构造方法（已定义其他构造方法则类将不会提供默认无参构造方法）

  • 构造方法不一定全为 public ，私有构造器的使用场景：

    如果想要在返回对象引用之前做一些额外的工作或者想记录到底创建了多少个对象（可能要限制其数量），用私有构造函数加上静态的成员函数的方法是大有裨益的 —— Java 编程思想

    私有构造方法+静态成员函数调用 可以用来实现单例模式和工厂方法模式，具体见Java 私有构造函数的应用

3. 共享设计模式

在JVM的底层实际上会存在一个对象池（不一定只保存String对象）当采用直接赋值的方式创建一个String类对象时，会将该对象入池保存，而后如果后续还有其他String类对象也采用直接赋值的方式，且设置了同样内容的时候，那么将不会再开辟新的堆内存空间，而是使用已有的对象进行为其分配引用，从而继续使用。
